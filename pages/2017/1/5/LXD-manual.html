Getting Started with LXD

  1. 创建第一个容器

    2. 假设使用ubuntu作为Ubuntu云镜像的别称，可以使用下面的命令创建你的第一个容器；

    lxc launch ubuntu my_container  # my_container is the name of container
lxc launch ubuntu # creat a container with a random name
lxc list  # show all containsers

    现在，你的第一个容器已经运行起来了，接下来我们可以使用容器里面的SHELL做一些事情；

    # 进入my_container的bash shell
lxc exec my_container -- /bin/bash
# 不进入my_container，直接运行一条命令
lxc exec my_container --apt-get update
# 获取/修改容器内的文件使用pull/push
lxc file pull my_container/etc/hosts .
lxc file push my_file my_container/tmp/
# 停止/删除容器
lxc stop my_container
lxc delete my_container
