    解决方法：

    方法2：.h中实现的函数加上static

  2. Java语法：import

    3. import:导入包中的类，语法如下：

    import package1.package2.classname;

    package为包名，classname为类名。例如：

    import java.util.Date;     //导入java.util包下的Date类
    import java.swing.*;       //导入java.swing包下的所有类

    注意:import只能导入包所包含的类，而不能导入包。

  5. Atom添加Git Plus插件

    6. Git Plus插件可实现在Atom编辑器里做git things，而不用切换至terminal。
    注意：本文基于ubuntu环境系统。安装步骤如下：

    • 配置user.name和user.email比较简单不做介绍。
    • 生成新的SSH Key，并添加到ssh-agent。可参考Git Help

      • 验证是否添加成功可以在终端执行ssh -T git@github.com进行查看。
    • 安装Git Plus package。在Atom->Settings,install项里面，搜索git-plus，安装即可。
    • 安装完成后，即可在Packages->Git Plus里面使用相应的git指令，亦或使用Ctrl-Shift-H快捷键，做一些git things。

    原文地址：git-plus

    在这个过程中可能遇到的问题及解决方法，如下所示：

    问题１：执行git push后，报错：fatal: could not read Username for 'https://github.com': Device not configured

    原因就是不能识别远端的用户名，解决方法就是重新设置远程地址即可。git remote set-url origin git@github.com:{yourname}/{your_repo_name}.git

    另外在Git 2.0中push有两种模式可选matching和simple。matching模式是提交名字一样的分支，所以可能的结果就是把其他分支的代码一起提交了，而这可能不是你想要的。而simple是push当前分支的代码。
    其设置命令如下:
    git config --global push.default simpleorgit config --global push.default matching

  6. Linux系统4G上网卡的使用配置

    7. 1、Linux系统下通过修改内核设置usbserial模式

    2、利用usb_modeswitch工具切换usb工作模式

    3、利用wvdial工具配置上网卡参数

    4、启用上网卡

    sudo insmod /lib/modules/$(uname -r)/kernel/drivers/usb/serial/usbserial.ko
sudo usb_modeswitch -W -c /etc/usb_modeswitch.conf
sudo wvdialconf /etc/wvdial.conf
sudo wvdial
